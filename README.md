# minesweeper-py
A pythonic version of old game minesweeper

# Steps to Launch
1. Clone the repo using this command - `git clone git@github.com:AakankshaDC/minesweeper-py.git`
2. cd into the repo
3. run `python3 app.py` - no external libraries needed!!!
